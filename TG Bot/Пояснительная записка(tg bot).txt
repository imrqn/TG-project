Задумка бота - конвертер валют.
При вводе команды /start(активируется функция main) бот приветствует пользователя и предлагает узнать курс валют.
Вместо клавиатуры появляются reply кнопки.
Нажимая кнопку "Перейти на сайт биржи"(обрабатывает get_text_messages) бот присылает сообшение с ссылкой на московскую биржу. 
При выборе кнопки "Узнать курс валют"(та же функция) бот присылает соообщение с Inline кнопками при нажатии на котрые(срабатывает callback_message) можно получить актуальную информацию о курсах валют. Если в предложенном перечне отсутствует необходимая вам валюта вы можете нажать на кнопку "Укажите сами" после чего бот даст вам определённые инструкции.
При вводе команды /news(функция your_curr отвечает за рассылку) бот предлагает подписаться на его рассылку в которой будет указываться инормация о валютах в зависимости от выбранного вами периода и времени. Чтобы отписаться от рассылки необходимо повторно ввести команду /news.
При каждом запросе в файл text1.txt(функция write_req) записывается точная дата, имя человека, а также его сообщения, исключительно в целях безопасности.
1. requirements.txt - есть
2. ORM модели - нет
3. работа с контекстом пользователя - есть (отправка курса валют)
4. загрузка и использование файлов: картинки, музыка - есть(отправление иконок погоды)
4. Использование стороннего API - есть(coinbase.com)
5. Хранение данных txt/csv - есть(запись данных запросов в файл text1.txt)
